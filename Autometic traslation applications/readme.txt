Library 01: "Goslate"
--> Python library and free Python API to google translation service
--> Quering Google translation
--> free, fast and simple
--> pip install goslate
--> import goslate

Library 02: "Translate"
--> Powerful translation tool with support for multiple translation providers.
--> Offers integration with Microsoft Translation API, Translated MyMemory API, LibreTranslate and more.
--> pip install translate
--> from translate import Translator

Library 03: "TextBlob"
--> Python library for processing textual data
--> Provides a simple API for diving into common NLP tasks such as classification, translation, and more
--> pip install textblob
--> from textblob import TextBlob

Other libraris we can use for translation: py-translate, Googletrans
